# Contributing to Octo

Thank you for your interest in contributing to Octo!

## Development Setup

1. Fork and clone the repository
2. Install dependencies: `pnpm install`
3. Download Geist fonts (see SETUP.md)
4. Create placeholder assets
5. Start development: `pnpm start`

## Code Style

- Use TypeScript for all new code
- Follow React Native best practices
- Use functional components with hooks
- Follow the existing code style and formatting
- Use Geist Sans for UI text, Geist Mono for code/technical text
- Use Lucide icons as primary, Octicons for Git-specific icons

## Design Guidelines

- Follow the dark theme color palette (see `constants/Colors.ts`)
- Maintain consistency with PixelPlayer's design principles
- Ensure proper spacing and typography hierarchy
- Use GitHub-inspired colors and styling

## Pull Request Process

1. Create a feature branch
2. Make your changes
3. Test thoroughly
4. Submit a pull request with a clear description

## Questions?

Feel free to open an issue for questions or discussions.
